# 欢迎来到 create-develop-framework 👋

使用其他语言阅读: [English](./README.md) | 简体中文

## 介绍

一个快速创建前端开发框架的脚手架的工具。

## 功能

  1. 交互式的创建前端开发框架.
  2. 支持以下项目类型
      - vue2-element-admin
      - vue3-element-admin
      - large-screen-visualization-echarts

## 安装
```bash
npm install create-develop-framework -g
```
## 使用
```bash
create-develop-framework create
```
